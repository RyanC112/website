54.Which of the following from fields are round buttons in a group of mutually exclusive pre-determined choices?

A. Checkboxes

B. Select lists

C. Radio buttons

D. Submit buttons

                                     

CheckBoxes

Checkboxes are a group of square shaped selectors/buttons that are checked off when selected hence the name checkbox. They allow for mutiple pre-determined answers to be selected at once.





Select List

A select list creates a drop down menu with a list of preset answers. Select list only allows for one answer to be selected.


Radio Button

Radio buttons are a group round button that only allow one pre-determined options be selected at the same time.




Submit Button

A submit button submits all form values to a form-handler. The Submit button is only allowed to submit the data put into the given form. The form-handler is typically a server page with a script for processing the input data.
