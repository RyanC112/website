Checkboxes are square shaped selectors/buttons that are checked off when selected hence the name checkbox. They allow for mutiple answers to be selected at once.





A select list creates a drop down menu with a list of preset answers. Select list only allows for one answer to be selected.


Radio buttons are a round button that Only allow one button in a group can be selected at the same time.




A submit button submits all form values to a form-handler. The form-handler is typically a server page with a script for processing the input data.
